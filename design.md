# BitsBreeze Website Design Style Guide

## Design Philosophy

**Scientific Elegance**: Modern, clean aesthetic that reflects cutting-edge scientific research while maintaining professional credibility. The design balances technical sophistication with accessibility, avoiding overly complex visuals that might alienate non-technical audiences.

**Innovation-Focused**: Visual language that conveys breakthrough technology and forward-thinking approach, using contemporary design patterns and subtle tech-inspired elements.

**Trust & Authority**: Professional color palette and typography that establish credibility in the scientific community while remaining approachable for potential partners and investors.

## Color Palette

**Primary Colors**:
- Deep Navy (#1a2332) - Primary brand color, conveying trust and scientific rigor
- Warm Copper (#c77d3a) - Accent color representing energy and innovation
- Soft Gray (#f8f9fa) - Background and neutral areas
- Pure White (#ffffff) - Clean backgrounds and text contrast

**Secondary Colors**:
- Muted Teal (#4a7c7e) - Supporting color for data visualization
- Light Copper (#e8b98a) - Lighter accent for hover states
- Charcoal (#2d3748) - Dark text and secondary elements

## Typography

**Display Font**: "Inter" - Modern, highly legible sans-serif for headings
- Large headings: 48px, bold weight
- Section headings: 32px, semibold weight
- Subsection headings: 24px, medium weight

**Body Font**: "Source Sans Pro" - Clean, readable for extended text
- Body text: 16px, regular weight
- Small text: 14px, regular weight
- Captions: 12px, medium weight

## Visual Language

**Geometric Precision**: Clean lines, precise alignments, and mathematical proportions that reflect scientific methodology

**Subtle Depth**: Minimal use of shadows and layering to create visual hierarchy without overwhelming the content

**Data-Driven Aesthetics**: Incorporation of graph-like elements, grid patterns, and scientific visualizations as design motifs

## Visual Effects & Animations

**Background Effects**:
- Subtle particle system using p5.js representing molecular activity
- Gentle gradient animations suggesting energy flow
- Geometric pattern overlays with low opacity

**Text Effects**:
- Typewriter animation for key headlines using Typed.js
- Color cycling emphasis on important terms
- Subtle glow effects on interactive elements

**Interactive Animations**:
- Smooth hover transitions with 3D tilt effects
- Timeline node animations with connecting line draws
- Card flip animations for research publications
- Parallax scrolling for depth perception

**Library Usage**:
- **Anime.js**: Smooth element animations and transitions
- **p5.js**: Particle background effects and data visualizations
- **ECharts.js**: Interactive research data charts
- **Typed.js**: Dynamic text animations
- **Splitting.js**: Advanced text effects for headings
- **Splide.js**: Image carousels for research galleries

## Layout Principles

**Grid System**: 12-column responsive grid with consistent spacing
**Whitespace**: Generous spacing to create breathing room and focus
**Hierarchy**: Clear visual hierarchy through size, color, and positioning
**Flow**: Logical content flow that guides users through the scientific narrative

## Component Styling

**Navigation**: Clean, minimal header with subtle transparency
**Cards**: Soft shadows with rounded corners, hover lift effects
**Buttons**: Solid fills with smooth hover transitions
**Forms**: Clean inputs with focus states and validation feedback
**Data Visualizations**: Consistent color usage with clear legends and labels

## Responsive Design

**Mobile-First**: Optimized for high-resolution displays with scalable elements
**Breakpoints**: 
- Mobile: 320px - 768px
- Tablet: 768px - 1024px  
- Desktop: 1024px+

**Adaptive Elements**: Timeline becomes vertical, technology visualizer simplifies, publication grid adjusts to single column
